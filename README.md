# Simple sleep

# No garbage code, No messages, No notifications, No time manipulations, No configs

An ultra-minimalist Minecraft datapack that enables multiplayer night skipping. If at least one player sleeps, the night is skipped for everyone.

## Installation
1. Download the `.zip` from releases
2. Place it into your world's `datapacks` folder
3. In-game, type `/reload` (or restart your server)

## Compatibility
*   Works on **1.20.x and all newer versions**
*   Compatible with everything. It only touches one specific game rule on load
