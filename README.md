# Simple sleep

# No Garbage code, No Messages, No Notifications, No Time Manipulations, No Configs

A ultra-minimalist Minecraft datapack that enables multiplayer night skipping. If at least one player sleeps, the night is skipped for everyone.

## Installation
1. Download the `.zip` from releases
2. Place it into your world's `datapacks` folder
3. In-game, type `/reload` (or restart your server)

## Compatibility
*   **Minecraft Versions:** Works on **1.17 and all newer versions**
*   **Server Software:** Compatible with Vanilla, Fabric, Quilt, Paper, and Purpur
*   **Other Datapacks:** Compatible with everything. It only touches one specific game rule on load
